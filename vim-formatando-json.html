[vim] Formatando JSON

// tags desenvolvimento dicas vim

Esse post inaugura uma nova seção aqui no blog. Regularmente postarei dicas gerais e diretas. A ideia é servir mais como um caderno de anotações pra mim mesmo em consultas futuras mas que também pode ser útil pra mais alguém.

A primeira dica e bem simples. Como formatar JSON utilizando o vim:

:%!python -m json.tool

O comando acima irá executar o módulo json.tool do Python.

O mesmo módulo pode ser utilizado fora do vim:

$ echo '{"atributo": "valor"}' | python -m json.tool
{
    "atributo": "valor"
}
