Sobre

Sou apaixonado por desenvolvimento de software e, desde que descobri isso, não consegui mais parar de programar. Em 2012 concluí minha graduação como Tecnólogo em Análise e Desenvolvimento de Sistemas pela Faculdade de Tecnologia de Sorocaba (FATEC) com rendimento máximo em disciplinas como: programação orientada a objetos, sistemas operacionais, algoritmos e lógica de programação, bancos de dados, estrutura de dados, dentre outras.

Tenho experiência profissional em desenvolvimento de soluções corporativas com tecnologia Java para ambiente web. Nesse ramo já trabalhei com ferramentas como JSP, Servlets, VRaptor, Spring, Struts, Hibernate, Tomcat e Jetty. Comumentemente preciso trabalhar com HTML5, CSS, Javascript (jQuery), Twitter Bootstrap, e me sinto completamente confortável com isso.

Possuo enorme interesse pela linguagem Python e grande parte de seu ecossistema. Ultimamente venho estudando-a cada vez mais e me aventurando em programação com essa linguagem (veja meu github); em especial com a utilização do framework web Django. Inclusive, o meu trabalho de conclusão de curso na faculdade envolvia justamente Python e Django. Como autêntico ser curioso e autodidata também já brinquei um pouco com Ruby on Rails, PHP, CodeIgniter, Shellscript e algumas outras tecnologias.

Desde que entrei na área de desenvolvimento de software venho trabalhando com equipes que se autodenominavam "ágeis". Devido à isso, desde os tempos de faculdade, estudei bastante conceitos de Desenvolvimento Ágil, Scrum, Extreme Programming e os diversos conceitos e boas práticas de desenvolvimento como TDD, Continuous Delivery, Pair Programming, etc. Tirando o Scrum, ainda não tive a oportunidade de trabalhar em algum ambiente que aplicasse de fato essas práticas, mas quem sabe um dia...
